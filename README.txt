Prerequisites:
Before you can run the tests, make sure you have the following installed:

Java (JDK 8 or above) - Download JDK
Apache Maven - Download Maven
Eclipse IDE (or any Java IDE) - Download Eclipse
Download ChromeDriver

1.  Clone the project and install it.
web URL: https://github.com/Lakshminarayananrqa/yahooFinance.git

2. Ensure that all the dependencies listed in the POM file are downloaded. 

3. To run the test, right-click on "TestRunner.java" and select "Run as JUnit Test."
reposetery path : src\test\java\testRunner\testRunner.java

4.The report will be generated at the following path and will capture error messages and screenshots.
Report Path : target\cucumber-html-report\index.html

5.The logs will be displayed in the console.
